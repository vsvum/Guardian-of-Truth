# 🛡️ Guardian-of-Truth

> **Система выявления галлюцинаций у больших языковых моделей**

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Offline](https://img.shields.io/badge/Работает-Оффлайн-brightgreen)](#)

`Guardian-of-Truth` — это легковесный оффлайн-инструмент на Python для анализа ответов больших языковых моделей (LLM) и оценки вероятности наличия в них галлюцинаций (вымышленных фактов). Не требует API-ключей и подключения к внешним сервисам.

---

## ✨ Особенности

- 🔍 **Анализ в реальном времени**: введите вопрос → получите оценку риска
- 🎚️ **Трёхуровневая шкала риска**:
  - 🟢 Низкий — ответ вероятно достоверен
  - 🟡 Средний — требуется дополнительная проверка
  - 🔴 Высокий — высокая вероятность галлюцинации
- 🔒 **Полная приватность**: вся обработка происходит локально, без отправки данных в облако
- 🎨 **Удобный CLI-интерфейс**: на базе библиотеки `rich` с анимацией, цветами и прогресс-барами
- 🐛 **Режим отладки**: подробный вывод метрик скоринга по запросу `/debug`
- 🚀 **Минимум зависимостей**: только `rich` для красивого вывода
